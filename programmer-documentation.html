How to Add a New Topic

To add a new learning topic, open solar-data.json and add another object inside the topics array. 

{
    "title": "Saturn",
    "description": "Saturn is famous for its large rings made of ice and rock.",
    "image": "🪐",
    "audio": "Text-to-speech audio is generated by the app."
}

Each topic must include a title, description, image, and audio field. The JavaScript file automatically reads the JSON file and adds the topic to the dropdown menu.

How the App Works

The JavaScript fetches the JSON file, loops through each topic, and adds the topic titles to the dropdown menu. When the user selects a topic, the page updates with the correct title, description, and image. The Read Aloud button uses the Web Speech API.
