Solar System Explorer

An educational app for learning about planets.

About the App

Solar System Explorer is a Progressive Web App that teaches users about planets using a menu, educational text, images, and text-to-speech audio.

App Preview

The app includes a dropdown menu, planet facts, a visual planet display, and a button that reads the content aloud.

🌍 🪐 🔴

How to Download

  1. Open the Solar System Explorer app page.
  2. Use Chrome, Edge, or another browser that supports PWAs.
  3. Click the install button when it appears.
  4. The app can then be opened like a regular app.
