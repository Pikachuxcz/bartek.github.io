How to use Solar System Explorer

Features

  • Learn about planets in the solar system.
  • Select topics from a dropdown menu.
  • View educational descriptions.
  • Use the Read Aloud button to hear the content.
  • Install the app as a Progressive Web App.
  • Use the app offline after it has been loaded once.

How to Use the App

  1. Open the Solar System Explorer app.
  2. Select a planet from the dropdown menu.
  3. Read the description shown on the screen.
  4. Click Read Aloud to hear the planet information.
  5. Click Install App if the browser gives the option.

Installation

To install the app, open it in a browser that supports PWAs. When the install option appears, click the Install App button. The app can then be launched from your device like a normal app.
